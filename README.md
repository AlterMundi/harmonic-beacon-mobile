## ğŸš€ Getting Started

### Prerequisites
-   Node.js v20+
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/harmonic-beacon.git
    cd harmonic-beacon-mobile
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App

You can run the app on an Android Emulator, iOS Simulator (Mac only), or directly on your physical device using the Expo Go app.

```bash
# Run on Android Emulator
npm run android 

# OR Run on iOS Simulator (Mac only)
npm run ios

# OR Run on physical device (scan QR code)
npm start
```
